# map、multimap容器

## 1 map基本概念

简介
> map中所有元素都是pair
> pair中第一个元素为key(键值)，起索引作用，第二个元素为value(实值)
> 所有元素都会根据元素自身的键值自动排序

本质
map/multimap属于关联式容器，底层结构二叉树实现  

优点  
可根据key快速找到value  

两者区别：允许有重复key值元素，不允许有重复key值元素
