# set/multiset容器

***集合类容器：所有元素都会在插入时被自动排序***  

## 1 set基本概念

本质：属于关联式容器，底层由二叉树实现  

set/multiset  
> set不允许容器中有重复的元素  
> multiset允许容器中有重复的元素  
包含头文件时，只包含set就行
