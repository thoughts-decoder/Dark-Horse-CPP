# vector 容器

## 1 vector基本概念

vector数据结构和 ***数组*** 非常相似，称为 ***单端数组***  
只能在数组尾端push pop  

区别：数组是静态空间，而vector是 ***动态扩展***  
动态扩展：不是在原空间之后续接新空间，而是找更大的内存空间，然后将原数据拷贝新空间，释放原空间。

> vector容器的迭代器是支持随机访问的迭代器
